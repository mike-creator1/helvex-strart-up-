How credits work

no surprises, no rounding tricks

01 · Per call

Tokens × model rate

Each request bills the exact prompt + completion tokens at the model's published rate. No tier bands, no rounding up.

02 · Per plan

Monthly cap refreshes

Your plan grants a monthly credit pool. Unused credits don't carry over; top-up packs do, and they never expire while you're active.

03 · Per second

Logged with trace ID

Every debit lands in the audit log with the request's trace ID — so you can always reconcile the bill back to the exact call.
