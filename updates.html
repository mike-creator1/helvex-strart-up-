Release · Nov 12, 2026 2.0 Prometheus

Release · Major version

Prometheus 2.0 — agents that run for hours.

The biggest update to our agentic model since launch. Plan-and-critique now runs as a default. Deterministic replay across multi-hour runs. New approval-gate primitives so humans stay on the irreversible decisions.

  • ▸ 4–8× longer continuous run horizon over 1.x
  • ▸ Native sub-agents for parallel branches
  • ▸ Approval gates on tool calls, file writes, sends

Release · Nov 12 v2.0

Prometheus 2.0 is live

Agents that run for hours, plan-and-critique by default, deterministic replay, and human-in-the-loop gates on irreversible steps. See the release post for the full breakdown.

Improvement · Nov 5 workspace

Spaces v2: live presence + threaded comments

Model · Oct 28 ether

Ether sub-100ms median, worldwide

Improvement · Oct 22 knowledge

Audio & video citation timestamps

Citations from audio and video sources now link to the exact second the answer came from. Click a citation, the player opens at that moment. Speaker diarization included.

Model · Oct 14 nexus

Cross-search across agent traces
