Release · Nov 12, 2026 2.0 Prometheus

Release · Major version

Prometheus 2.0 — agents that run for hours.

The biggest update to our agentic model since launch. Plan-and-critique now runs as a default. Deterministic replay across multi-hour runs. New approval-gate primitives so humans stay on the irreversible decisions.

  • ▸ 4–8× longer continuous run horizon over 1.x
  • ▸ Native sub-agents for parallel branches
  • ▸ Approval gates on tool calls, file writes, sends
