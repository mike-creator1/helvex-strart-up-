01 · Chat

A single thread that escalates.

Chat is where most of the work begins. Type, talk, or drop a file. The model picks itself based on what the message actually needs — or you pin one. When the conversation outgrows speed, escalate to depth or action without losing a single line of context.

Every chat is searchable, forkable, exportable, and shareable. Voice replies, image understanding, and inline file attachments are first-class — not features hiding behind menus.

What it gives you

  • →
    Mid-thread model switching. Press ⌘E to escalate Ether → Nexus → Prometheus. Context follows.
  • →
    Voice in 14 languages. Native voice input and output, with speaker timing preserved.
  • →
    File attachments up to 100 MB. PDF, DOCX, CSV, audio, video, code — all inline, all native.
  • →
    Image understanding. Charts, diagrams, screenshots, and handwriting all read accurately.
  • →
    Forked threads. Try a different angle without losing the original. Both stay searchable.
  • →
    @-mentions to teammates. Pull a colleague into a thread without leaving it.

02 · Projects

Folders that remember.

A project is a folder that holds a body of work. Pin the sources once. Add a custom system prompt. Choose a default model. Every chat opened inside the project starts from there — no more pasting the same brief into ten different conversations.

The project keeps a rolling summary of recent activity. When you come back after a week, you don't reread fourteen chats — you read one paragraph and pick up.

What it gives you

  • →
    Pinned files (up to 1 GB). Pin documents, decks, sheets, and URLs. They auto-update when the source changes.
  • →
    Custom instructions. A system prompt and tone of voice scoped to the project, not to your account.
  • →
    Default model. Pick which model opens by default for chats inside this project.
  • →
    Folder-level memory. Persistent facts the model picks up automatically — customer names, jargon, preferences.
  • →
    Sharing controls. Private, team-visible, or Space-wide. Permissions per project, not per file.
  • →
    Templates. Clone any project as a starting point — keep the structure, swap the content.

03 · Agents

Send work into the background.

Describe an outcome, set the rules, walk away. Agents plan, take steps, ask for input only when they have to, and leave a deterministic trace you can replay step by step. A 90-minute investigation runs once, repeats forever.

Every run is auditable. Pause and resume from any step. Edit the plan mid-run. Replay a past run identically when something needs explaining. Approval gates keep humans on the irreversible decisions.

What it gives you

  • →
    Plan-and-critique loop. The agent grades its own plan and revises before executing.
  • →
    Sub-agents. Parallel branches for embarrassingly parallel work — research, comparisons, sweeps.
  • →
    Pause & resume. Step in, edit the plan, ask a question, then let it continue.
  • →
    Schedules. Daily, weekly, custom cron. The agent runs without you opening the app.
  • →
    Approval gates. Human-in-the-loop on irreversible steps. Sign-off before the email sends.
  • →
    Deterministic replay. "How did that happen?" becomes a one-click question.

04 · Knowledge

Your team's library, asked plain.

Drop in everything your team has written, recorded, slid-decked, or scanned. HelveX indexes the lot — text, tables, audio, video, handwriting. Ask anything in natural language. Every answer cites the exact paragraph, slide, or audio timestamp it came from.

Permission-aware. Cross-language. Stale-content flagging when sources change in ways that affect past answers. New hires onboard against the library, not against your colleagues' calendars.

What it gives you

  • →
    Multi-format ingestion. PDF, DOCX, MD, CSV, slides, sheets, audio, video, scans — all native.
  • →
    OCR for scans. Handwriting and printed scans read accurately into searchable text.
  • →
    Audio & video transcription. With speaker diarization. Citations link to the exact timestamp.
  • →
    Permission-aware retrieval. Only surfaces what each viewer can see. Sensitive sources stay sensitive.
  • →
    Hybrid search. Semantic for meaning, keyword for exact phrases, numbers, and code.
  • →
    Stale-content flagging. When a source updates in a way that affects an old answer, the answer is flagged.

05 · Spaces

Where teams think together.

A Space is a shared room — chats, projects, agents, and a knowledge library that the team co-owns. Comment on any line, mention each other, hand off context with a click. New members get the full conversation history on day one, not after three months of catching up.

Roles, audit logs, and per-Space billing make Spaces ready for real teams — not just demo. Real-time presence shows who's reading what right now.

What it gives you

  • →
    Role-based permissions. Owner, editor, commenter, viewer. Per Space, not per file.
  • →
    Real-time presence. See who's reading the same chat or project right now.
  • →
    Threaded comments. On any message, agent step, or document. Mentions notify by email + in-app.
  • →
    Handoff with context. Pass a thread to a teammate; they get everything you saw plus your note on what to do next.
  • →
    Activity feed. Filterable by person, project, agent. The "what changed today" view.
  • →
    Per-Space audit log. Every action, every actor, exportable to your security team.

06 · History

Conversations stop being throwaway.

Every chat, every agent run, every draft is searchable — semantically and by keyword. Type the thing you remember, get the exact moment back. Filter by date, model, project, person, or content type. Decision archaeology becomes a single query, not a three-day excavation.

Pin moments to a Pins board. Resume a thread from any past message. Export to Markdown, PDF, or share as a link. Find the line where someone said the thing — even when the line is buried in a 40-minute call transcript.

What it gives you

  • →
    Semantic + keyword search. Find by meaning, fall back to exact phrase, number, or code snippet.
  • →
    Filters that matter. Date, model, project, person, content type, has-citation.
  • →
    Timeline view. Scroll your team's last six months chronologically.
  • →
    Resume from a result. Open a past moment and continue the thread from there.
  • →
    Cross-search agent traces. Find the exact moment in a 90-minute autonomous run.
  • →
    Find similar. Given a chat, surface the three most similar past chats.
