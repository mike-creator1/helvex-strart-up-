Best practices

  • 01Enable 2FA before you brief your first project. It locks the workspace before there's anything worth stealing.
  • 02Link a backup sign-in method. A connected provider survives a forgotten password without involving support.
  • 03Rotate API keys on a schedule, not just when something goes wrong. Every key change is zero-downtime.
  • 04Use scoped integrations. Give a connector only the permissions the workflow needs, never workspace-wide.
  • 05Review the audit log after onboarding new members. Trust and verify.
