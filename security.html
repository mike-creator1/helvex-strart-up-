Workspace controls

Data residency

Swiss-only

All data — prompts, embeddings, logs, model weights — stays in EU-CH-1. No US, no UK, no shadow replication.

Encryption

At rest & in transit

AES-256 at rest, TLS 1.3 in transit, per-tenant KMS envelope keys. Decryption never crosses the region boundary.

Audit

Every action logged

Sign-ins, key rotations, project briefs, plan changes — all recorded with actor, IP, and timestamp in Activity.

Best practices

  • 01Enable 2FA before you brief your first project — it locks the workspace before there's anything worth stealing.
  • 02Rotate API keys on a schedule, not just when something goes wrong. Every key change is zero-downtime.
  • 03Use scoped integrations — give a connector only the permissions the workflow needs, never workspace-wide.
  • 04Review Activity after onboarding new members. Trust and verify.
