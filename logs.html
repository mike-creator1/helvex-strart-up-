How logs work

structured, indexed, searchable

Structured

JSON, not strings

Every event is structured JSON with model, tokens, latency, trace ID, deployment, and the full payload. No grep — query.

Indexed

Trace ID first

The trace ID threads every call to the chat, agent, or workflow that triggered it — so you can always pull the full chain back.

Retained

30 days · longer on Scale

Logs roll off after 30 days on every plan. Scale extends this to 365 days and adds 7-year archive for audit shipping.
